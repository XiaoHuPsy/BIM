The data file **data_study3.mat** contains raw data of JOL (on a 0-100 continuous scale) and memory performance (0 or 1) for each trial.
The data file **data_study3_7bins.mat** contains raw data of JOL (on a 7-point discrete scale) and memory performance (0 or 1) for each trial.
- The **first column** is the subject ID.
- The **second column** is the experimental condition for each trial (1 = old, 2 = young).
- The **third column** is the JOL value for each trial.
- The **fourth column** is the memory performance for each trial.
