The data file **metaTrain_original_data.mat** contains raw data from Carpenter et al. (2019). See also the file **results.mat** at https://github.com/metacoglab/CarpenterMetaTraining/tree/master/Data.

Run *reorganize_metaTrain_data.m* to reorganize the data structure, and the reorganized data are saved in **metaTrain_organized_data.mat**.

Run *fit_metaTrain_data.m* to fit BIM to reorganized data, and the results are saved in **metaTrain_BIMresults.mat**.

The file **results_Pexp.sav** contains the fitted value of the parameter Pexp and can be analyzed in SPSS.
